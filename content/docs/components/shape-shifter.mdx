## Props

| Prop             | Type                             | Default                | Description                                                |
| ---------------- | -------------------------------- | ---------------------- | ---------------------------------------------------------- |
| `images`         | `string[]`                       | `sampleImages`         | Array of image URLs to display in the shape shifter        |
| `imagesInterval` | `number`                         | `500`                  | Interval between image transitions in milliseconds         |
| `shapeInterval`  | `number`                         | `1500`                 | Interval between shape transitions in milliseconds         |
| `shapeVariants`  | `ShapeShifterVariant[] \| false` | `defaultShapeVariants` | Array of shape variants or false to disable shape shifting |
| `className`      | `string`                         | `undefined`            | Additional CSS class names                                 |
