## Props

| Prop        | Type       | Default | Description                               |
| ----------- | ---------- | ------- | ----------------------------------------- |
| `lines`     | `string[]` | `[]`    | Array of text lines to display            |
| `delay`     | `number`   | `200`   | Delay between each word appearing (in ms) |
| `className` | `string`   | `""`    | Additional CSS class names                |
