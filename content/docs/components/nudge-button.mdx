## Props

| Prop        | Type                                     | Default | Description                              |
| ----------- | ---------------------------------------- | ------- | ---------------------------------------- |
| `children`  | `ReactNode`                              | -       | Content to display inside the button     |
| `nudgeText` | `string`                                 | -       | Text to display in the nudge tooltip     |
| `direction` | `'top' \| 'right' \| 'bottom' \| 'left'` | `'top'` | Direction the nudge tooltip appears from |
| `className` | `string`                                 | -       | Additional CSS class names               |
