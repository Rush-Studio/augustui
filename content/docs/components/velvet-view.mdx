## Props

| Prop     | Type          | Default | Description                    |
| -------- | ------------- | ------- | ------------------------------ |
| `slides` | `SlideItem[]` | `[]`    | Array of text lines to display |

### SlideItem

| Prop        | Type     | Default | Description                  |
| ----------- | -------- | ------- | ---------------------------- |
| `imageSrc`  | `string` | `""`    | Source URL of the image      |
| `imageText` | `string` | `""`    | Text to display in the image |
