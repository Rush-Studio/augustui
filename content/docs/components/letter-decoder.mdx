## Props

| Prop             | Type      | Default | Description                      |
| ---------------- | --------- | ------- | -------------------------------- |
| `className`      | `string`  | `""`    | Additional CSS class names       |
| `text`           | `string`  | `""`    | Text to decode                   |
| `triggerOnHover` | `boolean` | `false` | Trigger the effect on hover      |
| `triggerOnMount` | `boolean` | `true`  | Trigger the effect on mount      |
| `scrambleSpeed`  | `number`  | `50`    | Speed of the scramble effect     |
| `decodeTime`     | `number`  | `1000`  | Time it takes to decode the text |
